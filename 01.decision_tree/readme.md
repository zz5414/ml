### Decision tree 

- 구현 방향
	1. 데이터 셋 생성
		1. sklearn
		2. 직접 구현 (opencv로 클릭해서 생성하도록)
	2. 트레이닝
	3. 그래프 그리기
		1. mlxtend 라이브러리
		2. 직접 그래프 구현

- 구현 세부사항
	1. (1, 1), (2, 1), (3, 0) 의 데이터셋(x, y) 테스트
		1. sklearn
		2. 직접 구현
		3. Threshold가 2.5, Depth=1 인지 확인
